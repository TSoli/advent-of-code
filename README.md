# Advent of Code

Contains my solutions for [Advent of Code](https://adventofcode.com/).

## Languages

- 2023

  - C++ \(modern using C++20 standard\). Note
    [Eigen](https://eigen.tuxfamily.org/index.php?title=Main_Page) is required

- 2024
  - C++ (C++20 standard)

Build all C++ solutions with [cmake](https://cmake.org/) from the root directory

```
cmake build -S. -Bbuild
cmake --build build -j<num-jobs>
```

(Note: This must be rerun each time a new solution is added)
